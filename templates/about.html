🚗 Accident Prone Road Prediction

📌 Project Overview

This system uses Machine Learning to predict accident risk based on road conditions, weather, traffic density and other parameters.

🎯 Objective

To identify high risk roads and help reduce road accidents using data analysis.

🧠 Technology Stack

Python, Flask, Machine Learning, Pandas, Scikit-learn, HTML, CSS

📊 Dataset

20,000+ records including road type, weather, visibility, traffic conditions and accident severity.

🚀 Future Scope

Can be upgraded with real-time GPS tracking, live traffic APIs and mobile apps.
